🔧 Setup Guide — Run Local Backend

This frontend connects to a local Python server. Follow these steps once — then it runs forever.

1
Install Python & pip
Make sure Python 3.8+ is installed. Download from python.org if needed.
2
Install yt-dlp and dependencies
pip install yt-dlp flask flask-cors
3
Install ffmpeg (REQUIRED for 1080p+ audio merge)
ffmpeg handles merging video+audio streams automatically.
# Windows (via winget):
winget install ffmpeg

# macOS:
brew install ffmpeg

# Linux (Ubuntu/Debian):
sudo apt install ffmpeg
4
Save and run server.py
Save the server code (button below), then run it:
python server.py
5
Open index.html in your browser
Open this file in Chrome/Firefox. The status bar will turn green when connected. Downloads save to a downloads/ folder next to server.py.
